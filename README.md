## Project Overview

This project contains two Python programs:

1. Mathematical Calculator (MC)
2. Personal Pocket CGPA Calculator (PPC)

---

## 1. Mathematical Calculator (MC)

### Features

- Addition (+)
- Subtraction (-)
- Multiplication (*)
- Division (/)
- Floor Division (\)
- Exponential (^)
- Modulus (%)
- Clear (C)
- Exit (OFF)

The calculator accepts two numbers from the user and performs the selected arithmetic operation.

---

## 2. Personal Pocket CGPA Calculator (PPC)

### Features

- Accepts the number of courses.
- Accepts each course unit.
- Accepts each course score.
- Automatically determines the letter grade.
- Calculates Grade Points.
- Calculates Total Units.
- Calculates CGPA.
- Displays the corresponding Class of Degree.

### Grading System

| Score | Grade | Point |
|-------|-------|------|
| 70–100 | A | 5 |
| 60–69 | B | 4 |
| 50–59 | C | 3 |
| 45–49 | D | 2 |
| 40–44 | E | 1 |
| 0–39 | F | 0 |
